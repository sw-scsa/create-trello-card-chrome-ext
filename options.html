Trello Card Creator Settings

Trello API Configuration

To use this extension, you need to provide your Trello API credentials. Don't worry - these are stored securely on your device and never shared.

Get your API key from: https://trello.com/app-key
Generate a token by clicking the "Token" link on the API key page above.

How to Get Your API Credentials

1

Get Your API Key

Visit https://trello.com/app-key while logged into Trello

Copy the "Key" value and paste it in the API Key field above

2

Generate a Token

On the same page, click the "Token" link

Click "Allow" to grant the extension access to your Trello account

Copy the generated token and paste it in the API Token field above

3

Test and Save

Click "Test Connection" to verify your credentials work

Click "Save Settings" to store your configuration

Default Preferences

Privacy & Security

Your data is secure:

  • API credentials are stored locally on your device only
  • No data is sent to any servers except Trello's official API
  • This extension is open source - you can verify the code
  • You can revoke API access at any time from your Trello account settings
